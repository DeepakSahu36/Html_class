11. Complete List of Background-Related Attributes

Attribute Element(s) Purpose Status
background body, table, td, th Sets a background image Deprecated
bgcolor body, table, tr, td, th Sets background color Deprecated
text body Sets default text color Deprecated
link body Sets unvisited link color Deprecated
vlink body Sets visited link color Deprecated
alink body Sets active link color Deprecated

Important Notes

Note about the background attribute:

  • The background attribute accepts image URLs
  • It works on body, table, td, and th elements
  • Images tile/repeat by default (similar to CSS background-repeat: repeat)
  • No control over positioning, sizing, or attachment
  • Completely deprecated in favor of CSS background properties

Modern Alternative: Use CSS instead:

body {
    background-color: #f5f5f5;
    background-image: url('image.jpg');
    background-repeat: no-repeat;
    background-position: center;
    background-size: cover;
    color: #333333;
}

a {
    color: #0000ee;
}

a:visited {
    color: #551a8b;
}

a:active {
    color: #ff0000;
}
